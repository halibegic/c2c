C2C
======
C2C is the most important three-day event in the CEE region.

### How to use
* [Download](https://github.com/halibegic/c2c/archive/master.zip) C2C
* Type `npm i`

### Start server
* Type `npm run dev`

You should see a live browser at http://localhost:3000/.

### Production
* Type `npm run prod`
